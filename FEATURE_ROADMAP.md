# AOVault Feature Roadmap
*Updated: January 22, 2026*

## MVP - Phase 1 (Current Focus)

### Core Features
- [x] Import fics via URL (AO3)
- [x] Personal library view
- [x] Basic search
- [x] Favorites toggle
- [ ] **Branding update** - Warm gray/charcoal color scheme

### Enhanced Import & Organization
- [ ] Auto-download on save (local copy)
- [ ] Update detection (check if fic has new chapters)
- [ ] Pull description from source automatically
- [ ] Tag picker on import (show original tags, let user select)
- [ ] User custom tags

### Reading Experience
- [ ] Chapter tracking (where you left off)
- [ ] Chapters read count
- [ ] Read count (how many times you've read it)
- [ ] Reading history with dates

### Favorite Moments
- [ ] Highlight text to save excerpts (Kindle-style)
- [ ] Page number range selection
- [ ] Label/title your saved moments
- [ ] Up to 5 favorite moments per fic (max 2 pages each)

### Personal Organization
- [ ] Favorites list (max 24)
- [ ] Profile lock option for privacy
- [ ] AO3 username connection/sync

### Dashboard
- [ ] Most used tag
- [ ] Most read story
- [ ] Reading stats

## Design Direction

### Branding
- **Color:** Warm Gray/Charcoal (not AO3 red)
- **Logo:** AOVault wordmark in charcoal
- **Dark mode:** Default
- **Feel:** Cozy, bookish, candlelit library

### LiveJournal Nostalgia
- 2000s fandom aesthetic
- Message board vibes for prompts
- Personal journaling feel

---

## Technical Notes

### Auto-Download System
- Download fic content on save (not just metadata)
- Store locally for offline reading
- Check source URL periodically for updates
- Notify user when new chapters available

### Tag System
- Source tags (from AO3/FFN) - optional import
- User tags (personal organization)
- Display both types differently

### Reading Progress
- Track by chapter
- Track scroll position within chapter
- Sync across devices (future)
