## üì∏ Screenshot Requirements

### Technical Requirements
- **Format:** JPG or PNG (PNG recommended for better quality)
- **Color Space:** sRGB or Display P3
- **No Alpha Channel:** Screenshots must be fully opaque
- **Quantity:** 1-10 screenshots per device type
- **Order Matters:** First 3 screenshots are most important

### Content Requirements
- **No Placeholder Content:** Use real fanfiction data
- **Appropriate Content:** Follow App Store guidelines (12+ rating)
- **High Quality:** Sharp, clear, readable text
- **Consistent Style:** Same visual treatment across all screenshots
- **Localized:** If supporting multiple languages

## üé® Design Guidelines

### Visual Hierarchy
1. **Hero Content:** Make the main feature obvious
2. **Supporting Elements:** Don't clutter the screen
3. **Text Legibility:** Ensure all text is readable at small sizes
4. **Color Balance:** Use AO Vault brand colors consistently

### Brand Colors
- **Primary:** #E91E63 (Pink accent)
- **Background Light:** #FFFFFF
- **Background Dark:** #121212
- **Text Primary:** #212121
- **Text Secondary:** #757575

### Typography
- **Headers:** System Font Bold
- **Body:** System Font Regular
- **Accent Text:** System Font Medium
- **Minimum Size:** 11pt when viewed on device

### Device Frame Options
1. **No Frame:** Clean, maximum screen real estate (RECOMMENDED)
2. **Device Frame:** Shows iPhone/iPad frame around screenshot
3. **Marketing Frame:** Add promotional text above/below
